// 01

about

OrbitWatch is a SwiftUI project exploring how a curious mind can interface with NASA's public data — a quiet, focused app for keeping an eye on the ISS and the night sky.

Built around two pillars: a live 3D globe tracking the ISS via the open-notify API, and a daily APOD reader powered by NASA's image-of-the-day endpoint.

// 03

features

core capabilities
/01
Live ISS Tracker
Real-time orbital position rendered on a 3D MKMapView with custom annotation, latitude/longitude readout, and automatic 5-second polling — synced through Combine publishers.
/02
Picture of the Day
NASA APOD integration with title, captured date, and full-length explanation. Async image loading with cached fallback for offline reads.
/03
Two-tap navigation
Home → feature → detail. No tab bars, no side menus. The architecture stays out of the way.
/04
Edge-aware UI
Status bar overlays, safe-area-respecting controls, and gesture-driven map interactions optimized for one-handed use.
